Des modifications ont été apportées à la base de données.
Par conséquent, il faudra la réimporter dans phpMyAdmin sous peine de non fonctionnement de l'interface web.

Les fichiers relatifs à la base de données sont dans le dossier bdd.
Les fichiers à mettre dans le dossier www de uwamp sont dans le dossier Projet-BDD-Pokedex.

creation_bdd_pokedex.sql permet de créer la base de données pokedex avec toutes ses tables mais non peuplée. Pour la peupler, il faudra exécuter peuplement.py
